# Compare & Contrast Essay Website

This GitHub Pages site presents a fully English-language compare & contrast essay based on the original content from the referenced Google Sites page. The website includes:

- The original essay text (presented verbatim in English)
- Clean layout styled with `Times New Roman`
- Visual charts and images for illustration
- Fully static, easy-to-host `index.html` + `styles.css`

## File Structure
```
├── index.html
├── styles.css
└── README.md
```

## How to Deploy on GitHub Pages
1. Create a new GitHub repository.
2. Upload all files (`index.html`, `styles.css`, `README.md`).
3. Go to **Settings → Pages**.
4. Under *Build and Deployment*, choose:
   - Source: **Deploy from branch**
   - Branch: **main** → **/root**
5. Save.
6. Your site will be available shortly at:
   `https://<your-username>.github.io/<repository-name>/`

## Editing / Customizing
- Modify `index.html` to adjust wording, structure, or add new visualizations.
- Use `styles.css` to change colors, spacing, or typography.
- To replace charts, edit the `<img>` or `<svg>` tags inside `index.html`.
